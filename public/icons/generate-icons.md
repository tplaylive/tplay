# T PLAY App Icons

This folder contains the app icons for the T PLAY PWA. 

## Required Icon Sizes:
- icon-72x72.png (Android)
- icon-96x96.png (Android)
- icon-128x128.png (Android)
- icon-144x144.png (Android)
- icon-152x152.png (iOS)
- icon-192x192.png (Android, recommended)
- icon-384x384.png (Android)
- icon-512x512.png (Android, recommended)

## Icon Design Guidelines:
- Use the T PLAY gradient logo
- Ensure visibility on light and dark backgrounds
- Follow platform-specific guidelines
- Use high-quality PNG format
- Include proper padding for safe area

## Auto-Generation:
You can use online tools like:
- realfavicongenerator.net
- favicon.io
- Or design tools like Figma/Canva

## Current Status:
Icons need to be generated and placed in this folder for the PWA to work properly.
